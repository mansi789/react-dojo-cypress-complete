# Build and test an application using Cypress

Step 1:
Git clone the repo 
Step 2 :
In order to run this project, we need to have node and npm installed
Step 3 :
Cd into our project and run 
`npm install`
Step 4 :
Before doing that run `npm update`
Step 5 :
To run the app 
`npm run dev`
Step 6:
To run cypress
npm run cy:open
